Producer Service

Strategia Ibrida (Circuit Breaker)

  • Invio Asincrono: Kafka buffer + callback (HTTP 200).
  • Fallback Locale: Coda in-memory ‚Üí HTTP 202 (Se Kafka Down).
    • Thread background: Retry automatico ogni 3 secondi.
  • Backpressure: Coda piena ‚Üí HTTP 503.
src/producer/producer.py
def try_send_event(event): if producer: producer.send(TOPIC, event) return "200 OK" # Kafka Unreachable -> Fallback RAM try: local_queue.put_nowait(event) return "202 Accepted" except queue.Full: return "503 Unavailable"

Resilienza e Self-Healing

Design for Failure

Readiness Probe

Verifica connettivit√† Kafka/MongoDB. Se fallisce, rimuove il pod dal Load Balancer.

Liveness Probe

Heartbeat su file per il Consumer. Se il processo va in deadlock, K8s riavvia il pod.

Leader election automatica se un broker crasha. Il Producer switcha nodo trasparentemente.
